set up python virtualenv and various packages:

$ sudo easy_install virtualenv
$ virtualenv --no-site-packages /srv/sourcy/sourcy-env
$ . /srv/sourcy/env/bin/activate
$ pip install tornado lxml WTForms SQLAlchemy python-dateutil rdflib rdfextras MySQL-python py-bcrypt alembic PIL pylibmc


NOTE about installing PIL (on ubuntu 11.10/64bit):

pip doesn't install PIL properly under 64bit ubuntu - it misses zlib, libjpeg
and libfreetype.
Cheesy hack workaround is to install unbuntu PIL package then copy the files
into the virtualenv manually:

$ sudo apt-get install python-imaging
$ cp -r /usr/lib/python2.7/dist-packages/PIL {{VIRTUALENV}}/lib/python2.7/site-packages
